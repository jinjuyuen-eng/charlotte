# 게임 규칙

> "Catch the Fruits" - 하늘에서 떨어지는 과일을 받아라!

## 🎮 게임 제목
**Catch the Fruits** (과일 받기 챌린지)

## 📝 게임 설명
하늘에서 맛있는 과일과 위험한 폭탄이 떨어집니다!
여러분의 몸을 바구니라고 생각하고, 왼쪽, 오른쪽, 중앙으로 움직여 과일을 받아내세요.
폭탄을 피하지 못하면 점수가 깎이거나 게임이 끝날 수 있습니다.

## 🎯 게임 목표
- **60초** 동안 최대한 많은 과일을 받아 **최고 점수**를 획득하세요.
- 폭탄은 절대 받지 마세요!

## 🕹️ 게임 규칙

### 기본 규칙
1. **화면 구성**: 게임 화면은 **3개의 라인(Left, Center, Right)**으로 나뉩니다.
2. **플레이어 이동**: 포즈를 취해 바구니(플레이어)를 해당 라인으로 이동시킵니다.
3. **점수 획득**: 떨어지는 과일과 바구니가 겹치면 점수를 획득합니다.
4. **패널티**: 폭탄을 받으면 점수가 깎이거나 라이프가 줄어듭니다.

### 아이템 및 점수
| 아이콘 | 이름 | 점수 / 효과 |
|:---:|---|---|
| 🍎 | **사과** | **+10점** |
| 🍇 | **포도** | **+20점** |
| 🍊 | **오렌지** | **+30점** |
| 💣 | **폭탄** | **-50점 & 라이프 1 감소** |
| ⏰ | **시계** | **시간 +5초** (선택 사항) |

### 게임 오버 조건
1. **제한 시간 0초** 도달
2. **라이프 소진** (폭탄 3번 받음)

## 🎨 필요한 포즈 목록 (Teachable Machine 클래스)

게임 플레이를 위해 **3가지 포즈**를 학습해야 합니다.

| 클래스 이름 (Label) | 행동 (Action) | 설명 | 예시 포즈 |
|---|---|---|---|
| **LEFT** | 왼쪽 라인 이동 | 고개를 왼쪽으로 젖히거나,<br>왼손을 듭니다. | 👈 / 🙋‍♀️ (왼손) |
| **CENTER** | 중앙 라인 이동 | 정면을 바라보거나,<br>차렷 자세를 유지합니다. | 😐 / 🧍 |
| **RIGHT** | 오른쪽 라인 이동 | 고개를 오른쪽으로 젖히거나,<br>오른손을 듭니다. | 👉 / 🙋‍♂️ (오른손) |

> **Tip**: 모델 학습 시 `LEFT`, `CENTER`, `RIGHT` 대문자로 라벨을 설정하면 코드를 수정할 필요가 줄어듭니다!

## 📊 난이도 시스템 (레벨 디자인)
- **Start (0~20초)**: 과일이 천천히 떨어짐
- **Speed Up (20~40초)**: 떨어지는 속도 증가, 폭탄 등장 확률 증가
- **Fever (40~60초)**: 과일이 비처럼 쏟아짐! (점수 2배 찬스)

## 💡 개발 가이드
이 규칙을 바탕으로 코드를 구현할 때 참고할 사항입니다.

- **`gameEngine.js`**:
  - `commands` 배열 대신, **실시간 위치 제어** 방식이 필요합니다.
  - 기존의 "명령을 보고 따라하기" 로직을 → **"떨어지는 물체와 충돌 체크"** 로직으로 변경해야 합니다.
  - 포즈(`LEFT`, `CENTER`, `RIGHT`)가 인식될 때마다 플레이어 위치 변수(`playerX`)를 업데이트합니다.
  - `update()` 함수(게임 루프)를 추가하여 매 프레임마다 과일의 위치를 아래로 이동시켜야 합니다.

- **`index.html` / `style.css`**:
  - 3개의 세로 라인을 시각적으로 표시하면 좋습니다.
  - 플레이어(바구니)와 떨어지는 아이템을 DOM 요소(div)로 생성하여 CSS `top`, `left` 속성으로 움직입니다.
